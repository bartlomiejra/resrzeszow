== Description ==
Avadanta Companion plugin adds Extra sections and functionality to the Avadnata theme. This plugin for only Avadnata theme. Avadanta Companion is a plugin build for enhance the functionality of Avadanta WordPress Theme.

[Avadanta Theme](https://www.avadantathemes.com/product/avadanta-free-multipurpose-business-wordpress-theme/)

This plugin is the for Avadanta themes. It creates the slider section , service section, portfolio section and testimonial on home page in the Avadanta theme.
Avadanta is a plugin build to enhance the functionality of WordPress Theme made by Avadanta.

This plugin create repeater controls in the customizer settings allowing you to create a live site without moving out to customizer screen. Right now plugin have a support for Avadanta WordPress Theme. In future this plugin will support other themes by Avadanta.
